2 Die Gruppenverwaltung

2.1 Benutzergruppen Privilegien zuordnen

Um einer Benutzergruppe Privilegien zu zuordnen, wählen Sie die entsprechende Benutzergruppe in der Listenansicht Gruppe aus und aktivieren bzw. deaktivieren das gewünschte Privileg mit einem Haken vor dem jeweiligen Privileg. Klicken Sie anschließend auf die Schaltfläche Übermehmen, um die zugeordneten Privilegien für diese Gruppe zu übernehmen.

2.2 Benutzergruppen erstellen

Wollen Sie eine neue Benutzergruppe erstellen, geben Sie in dem Texteingabefeld Gruppenname die gewünschte Bezeichung für die Benutzergruppe ein, optional können Sie noch einen Kommentar in dem Texteingabefeld Kommentar eingeben, und klicken Sie dann auf Speichern, um die neue Benutzergruppe anzulegen.

2.3 Benutzergruppen löschen

Eine Benutzergruppe kann gelöscht werden, in dem sie in der Listenansicht Gruppe ausgewählt wird und anschließend über die Schaltfläche Löschen gelöscht wird. Eine einmal gelöschte Benutzergruppe lässt sich nicht wiederherstellen.

3 Mit Netzwerkcomputer verbinden

Der XP Usermanager bietet die Möglichkeit auch andere NT Maschinen* im Netzwerk lokal zu administrieren. Um sich mit einem Remotecomputer zu verbinden, rufen Sie den Dialog "Mit Netzwerkrechner verbinden" über den Menüpunkt "Computer wählen..." auf.

SteuerelementFunktion
Eingabefeld ComputerName oder IP-Adresse des Remotecomputers.
Schaltfläche NetzwerkRuft einen Dialog auf, um einem Computer im Netzwerk auszuwählen.
Eingabefeld BenutzernameBenutzername eines Administrators auf dem Remotecomputer
Kontrollkästchen SpeichernLegt fest, ob der zu letzt ausgewählte Remotecomputer und der zu letzt eingegeben Benutzername in der Registry gespeichert werden sollen.
Eingabefeld PasswortPasswort für das Remotebenutzerkonto.
Schaltfläche OKSchliesst den Dialog und es wird versucht sich mit dem ausgewählten Computer zu verbinden.
Schaltfläche AbbrechenSchliesst den Dialog und bricht den Vorgang ab.

*) Getestet unter Windows 2000 und Windows XP Professional. Windows XP Home lässt eine Remotadministration aufgrund der eingeschränkten Netzwerkunterstützung nicht zu.

4 Installation

4.1 Installation

Der XP Usermanager erfordert keine Installation. Die Dateien im Archiv können in ein beliebiges Verzeichnis entpackt werden. Das Programm kann direkt von dort gestartet werden.

4.2 Deinstallation

Um den XP Usermanager zu deinstallieren, führen Sie einfach die Stapelverarbeitungsdatei uninstall.bat aus. Diese entfernt alle Einträge für den Benutzer aus der Registry. Danach können sie einfach alle Programmdateien und den Programmordner löschen.
